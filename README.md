# Σύστημα Φωνητικών Εντολών - Mastronek AI

## 📋 Περιγραφή
Ένα σύστημα επεξεργασίας φωνητικών εντολών σε πραγματικό χρόνο με υποστήριξη Ελληνικής γλώσσας.

## 🚀 Λειτουργίες
- Αναγνώριση φωνητικών εντολών στα Ελληνικά
- Μετατροπή κειμένου σε ομιλία (TTS)
- Modular αρχιτεκτονική για προσθήκη νέων εντολών
- Συνεχής ακρόαση με multi-threading
- Διαχείριση περιβάλλοντος και μνήμης
